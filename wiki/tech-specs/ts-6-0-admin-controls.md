# TS 6.0 — Administrative Controls

## 6.1 — Responsibility

The Plant Manager or designee is responsible for the safe operation of the facility.

## 6.2 — Organization

### Shift Staffing

Minimum shift crew composition is specified for various modes of operation. Key positions include:
- Shift Manager (SRO license)
- Unit Supervisor (SRO license)
- Reactor Operators (RO license)
- Non-licensed operators

<div class="callout callout-exam">
<div class="callout-label">Key Exam Concept</div>
The minimum shift staffing requirements specify the number of licensed operators (RO and SRO) required for each mode of operation. An SRO must be present in the control room during reactor startups, fuel handling, and recovery from reactor trips.
</div>

## 6.8 — Procedures and Programs

Key programs required by TS administrative controls:

| Program | Purpose |
|---------|---------|
| Primary Coolant Sources Outside Containment | Minimize leakage from systems outside containment that could contain radioactive fluids during accidents |
| In-Plant Radiation Monitoring | Monitor radiation levels in areas normally accessible to personnel |
| Radioactive Effluent Controls Program | Control liquid and gaseous effluents per ODCM |
| Radiological Environmental Monitoring Program | Monitor the environment around the plant |
| Steam Generator Program | SG tube inspection, repair, and plugging criteria |
| Containment Leakage Rate Testing Program | Type A, B, and C testing per 10CFR50 Appendix J |
| Surveillance Frequency Control Program (SFCP) | Control surveillance frequencies; changes require NRC-approved methodology |

## 6.9 — Reporting Requirements

### Routine Reports
- Annual Radiological Environmental Operating Report
- Annual Radioactive Effluent Release Report
- Core Operating Limits Report (COLR) — submitted for each reload cycle

### Special Reports
- Required following specific events (e.g., ECCS actuation, safety limit violation)

## 6.16 — TS Bases Control Program

Changes to the Bases can be made without prior NRC approval provided the changes do not involve a change to the TS incorporated in the license. Changes shall be reviewed and approved by the onsite review function prior to implementation.
