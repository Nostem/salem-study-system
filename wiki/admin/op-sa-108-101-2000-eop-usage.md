# OP-SA-108-101-2000 — Emergency/Abnormal Operating Procedure Program

## Purpose

Administrative procedure governing the use of Emergency Operating Procedures and defining the meaning of EOP flowchart symbols.

## Key Actions / Information

<div class="callout callout-exam">
<div class="callout-label">Exam — 2023 Q23</div>
The Concurrent Step symbol means initiate the stated action while continuing with the existing flow path. Other EOP flowchart symbols:

- **Sequential Branching** — complete the branched action before continuing with the existing flow path
- **Conditional Step** — execute if/when the stated conditions occur
- **Wait Step** — wait for the stated conditions to be met before proceeding
</div>
