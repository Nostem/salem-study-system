# EP-SA-325-F2 — Initial Contact Message Form

## Purpose

Provides the standardized form for initial notification to offsite agencies (NRC, state, and local) following an emergency declaration. The ICMF must be accurately completed and transmitted within required timeframes.

## Key Sections

| Section | Content | Common Errors |
|---------|---------|---------------|
| I | Station identification, unit | — |
| II | Emergency classification, declaration time, event description | Time errors, wrong event description |
| III | Release status (in progress / not in progress) | Misidentifying release path status |
| IV | Meteorological data | — |
| V | Protective action recommendations | — |

## Event Description (Table 1 of EP-SA-325-102)

The event description must match the Table 1 barrier assessment terminology. For an SGTR, the correct description is "Loss of the Reactor Coolant System Barrier."
