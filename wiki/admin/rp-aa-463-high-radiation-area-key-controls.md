# RP-AA-463 — High Radiation Area Key Controls

> DRAFT — Reconstructed from exam materials

## Purpose

Provides the requirements and controls for key release and access into High Radiation Areas (HRA). Defines authorization levels for key release based on area dose rates and posting classifications.

## Key Actions / Information

### Key Release Authorization (Attachment 1)

| Area Classification | Authorization Required | Source |
|-------------------|----------------------|--------|
| High Radiation Area (HRA) | SRPT (Senior Radiation Protection Technician) | RP-AA-463 Att. 1 |

<div class="callout callout-jpm">
<div class="callout-label">JPM — 2019 SRO-A4</div>
HRA entry authorization for Unit 1 Pipe Alley (Elevation 84 ft, Auxiliary Building): area posting is <span class="hi-exam">High Radiation Area (HRA) and Contaminated Area (CA)</span>. Highest On Contact dose rate = <span class="hi-exam">205 mrem/hr</span>. Highest General Area dose rate = <span class="hi-exam">10 mrem/hr</span>. Per Attachment 1, <span class="hi-exam">SRPT authorizes key release for HRA entry</span>. Operators may also need to sign onto RWP-1 and receive an HRA brief from Rad Pro per RP-AA-460 P&Ls 2.1.1 and 2.1.3. Dose calculation: [205 mrem/hr x (1/60)] x 2 min = <span class="hi-exam">6.8 mrem</span>.
