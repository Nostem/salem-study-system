# CC-AA-112 — Temporary Configuration Changes

## Purpose

Administrative procedure governing the process for controlling temporary design/configuration changes at Salem and Hope Creek stations.

## Key Requirements

- **CRS** is responsible for approving installation of Temporary Configuration Changes (TCCs)
- Design Engineering Manager is an approver in the development of the TCCP but does not approve installation
- TCCs established during maintenance activity (MR90s) are not subject to a 50.59 review provided:
  - The MR90 is in direct support of maintenance
  - The MR90 will be installed for no more than 90 days at power
- TCCs installed greater than 90 days require a 50.59 review
