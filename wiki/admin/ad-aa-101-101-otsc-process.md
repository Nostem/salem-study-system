# AD-AA-101-101 — OTSC Process

> ⚠️ DRAFT — Reconstructed from exam materials

## Purpose

Provides requirements for implementing On-The-Spot Changes (OTSCs) to technical and implementing procedures, including criteria for what constitutes a change to the intent of a procedure.

## Key Actions / Information

<div class="callout callout-exam">
<div class="callout-label">Exam — 2022 Q69</div>
<span class="hi-exam">Altering an Inspection Hold Point</span> is considered a change to the INTENT of a procedure and must go through the formal procedure change process (cannot be done via OTSC). The following are NOT changes to intent and CAN be done via OTSC: increasing management oversight/approval level, adding clarifying text to a step, and correcting a step reference to the correct attachment.
