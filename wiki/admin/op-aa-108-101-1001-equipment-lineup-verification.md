# OP-AA-108-101-1001 — Equipment Lineup Verification

## Purpose

Provides requirements for checking manual valve positions, including throttled valve verification procedures.

## Key Actions / Information

<div class="callout callout-exam">
<div class="callout-label">Exam — 2023 Q67</div>
Checking Manual Throttle Valve Position (Section 4.4): When operation of a throttled valve is necessary to determine its position, consult with the appropriate Operations supervisor prior to manipulating. <span class="hi-exam">Check throttled valve position by counting the number of 360-degree turns required to fully close the valve, then re-position the valve by opening the same number of turns counted.</span> Restoring locked throttled valves requires concurrent verification. Personnel must be aware of mechanical valve operator slack.
</div>
