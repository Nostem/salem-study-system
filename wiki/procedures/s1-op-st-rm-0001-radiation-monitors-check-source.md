# S1.OP-ST.RM-0001 — Radiation Monitors Check Source

> ⚠️ DRAFT — Reconstructed from exam materials

## Purpose

Provides guidance for performing check source testing on radiation monitors at Salem Unit 1.

## Key Information

### 1R19A (SGBD Radiation Monitor) Check Source Response
- When the check source pushbutton is pushed, counts will RISE then RETURN to the original reading
- Actuation of interlocks is NOT expected during check source

### Other Radiation Monitors
- Some radiation monitors maintain elevated counts as long as the check source pushbutton is held
- Some radiation monitors may actuate interlocks during check source
