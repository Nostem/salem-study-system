# S1.OP-SO.WL-0002 — Release of Radioactive Liquid Waste (Unit 1 CCHX Path)

## Purpose

Provides direction for Unit 1 release of radioactive liquid waste from CVCS Monitor Tanks through the CC Heat Exchanger (CCHX) and Circulating Water (CW) system discharge path. Attachment 2 contains the Release Verification checklist (Section 2.2) used to determine the correct CCHX and CW pump alignment for the release.

## Key Actions / Information

### Section 2.2 — Release Verification

The CRS must verify:
1. Which CCHX is available for the release path
2. Which CW Pumps are in service to provide minimum required dilution flow
3. That the selected path meets the minimum dilution flow rate specified by Chemistry

The release path goes from the CVCS Monitor Tank through the selected CCHX to the CW system, where circulator (CW pump) flow provides dilution before discharge.
