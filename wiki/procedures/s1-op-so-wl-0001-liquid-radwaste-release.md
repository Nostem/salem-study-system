# S1.OP-SO.WL-0001 — Release of Radioactive Liquid Waste

## Purpose

Provides direction for release of radioactive liquid waste from CVCS Monitor Tanks through the CC Heat Exchanger and CW system discharge path.

## Key Actions / Information

<div class="callout callout-exam">
<div class="callout-label">Exam — 2023 Q61</div>
<span class="hi-exam">Only one circulator is required to be in service to allow a liquid release to continue.</span> The 1R18 radiation monitor does not isolate or close the 1WL51 overboard valve on loss of flow. Loss of one circulator does not automatically terminate the release.
</div>
