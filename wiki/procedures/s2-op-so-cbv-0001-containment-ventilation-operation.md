# S2.OP-SO.CBV-0001 — Containment Ventilation Operation

## Purpose

Provides direction for normal operation of containment ventilation systems including containment fan coil units (CFCUs).

## Key Actions / Information

<div class="callout callout-jpm">
<div class="callout-label">JPM — 2023 Sim-f</div>
Referenced during CFCU surveillance testing. CFCU flow acceptance criteria use P&L 3.4 — for a measured differential pressure, use the next higher DP value from the table to determine minimum required flow.
