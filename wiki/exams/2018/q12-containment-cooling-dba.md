<strong>Given:</strong><br>

1. ONLY ONE CS Pump operating<br>
2. ONLY ONE CS Pump operating plus ONLY 3 CFCUs operating<br>
3. ONLY ONE CS Pump operating plus ONLY 2 CFCUs operating<br>
4. ONLY 5 CFCUs operating<br><br>
Which ONE of the following completes the statements below?<br><br>
___ are the MINIMUM Containment Cooling components required to be in operation that will maintain Containment Pressure, Temperature and Humidity within design limits with a DBA occurring inside containment.
