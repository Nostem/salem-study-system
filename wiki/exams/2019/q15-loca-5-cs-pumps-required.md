<div style="padding:8px 12px;background:#052e16;border-left:3px solid #22c55e;border-radius:0 4px 4px 0;margin-bottom:10px;color:#c8cdd5;"><strong style="color:#22c55e;">✓ B. Correct.</strong> IAW 2-EOP-LOCA-5 Step 6 Table B, NO TRAINS of Emergency Recirculation are available. Consequently, the candidate must analyze 2-EOP-LOCA-5 Steps 6.1 through 9 and Table C to determine if any of the CS Pumps are required to be stopped. Based on Table C and Step 9 with Containment Pressure = 19 psig, RWST Level equal to 10 feet, and only 3 CFCUs running in slow speed (with 2B 4KV Vital Bus de-energized and locked out, ONLY 3 CFCUs are running in slow speed), 0 CS Pumps are required. Consequently, the crew will stop BOTH Containment Spray Pumps.</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">✗ A.</strong> The candidate could correctly analyze Table B and then incorrectly analyze Table C to determine that both CS Spray Pumps are required. This would then cause the candidate to incorrectly conclude that the crew's NEXT action will be to add makeup to the RWST (2-EOP-LOCA-5 Step 10).</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">✗ C.</strong> The candidate could correctly analyze Table B and then incorrectly analyze Table C to determine that only one CS Spray Pump is required. This would then cause the candidate to incorrectly conclude that the crew's NEXT action will be to stop ONLY ONE Containment Spray Pump (2-EOP-LOCA-5 Step 9).</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">✗ D.</strong> The candidate could incorrectly analyze Table B.
