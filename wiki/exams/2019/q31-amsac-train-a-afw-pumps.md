<strong>Given:</strong><br><br>
&bull; Unit 2 is at 45% Reactor Power<br><br>
At time 14:00<br>
&bull; A malfunction within AMSAC causes ONLY Train A of AMSAC to actuate<br><br>
Which ONE of the following completes the statement below?<br><br>
The actuation of Train A of AMSAC will send an AUTOMATIC start signal to ___ AFW Pump(s).
</div>

<div style="margin-bottom:14px;">
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>A.</strong> ALL</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>B.</strong> ONLY 21</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>C.</strong> ONLY 21 and 22</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>D.</strong> ONLY 21 and 23</div>
</div>

<div style="margin-top:10px;padding:16px;background:#0a0e17;border:1px solid #1a2035;border-radius:6px;font-size:12px;">
<div style="padding:8px 12px;background:#052e16;border-left:3px solid #22c55e;border-radius:0 4px 4px 0;margin-bottom:10px;color:#c8cdd5;"><strong style="color:#22c55e;">&#10003; D. Correct.</strong> The 21 AFW receives an AUTOMATIC start signal when AMSAC Train A actuates. The 22 AFW receives an AUTOMATIC start signal when AMSAC Train B actuates. The 23 AFW receives an AUTOMATIC start signal when either train of AMSAC actuates. Consequently, when ONLY Train A of AMSAC actuates, 21 and 23 AFW Pumps receive an AUTOMATIC start signal.</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">&#10007; A.</strong> Incorrect but plausible. The candidate could not understand how AMSAC circuitry works and incorrectly conclude that ALL AFW pumps receive an AUTOMATIC start signal.</div>

The candidate could not understand how AMSAC circuitry works and incorrectly conclude that ONLY 21 pump receives an AUTOMATIC start signal.</div>

The candidate could not understand how AMSAC circuitry works and incorrectly conclude that ONLY 21 and 22 AFW pumps receive an AUTOMATIC start signal.</div>
