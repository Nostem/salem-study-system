<span style="color:#60a5fa;font-size:14px;font-weight:600;">Q42 — Containment Spray Pump Start with SEC</span>
<span style="font-size:10px;padding:2px 8px;background:#0d1220;color:#6b7280;border-radius:3px;">026 K2.01 (3.4)</span>
</div>

<div style="font-size:13px;color:#9ca3af;margin-bottom:14px;line-height:1.7;">
<strong>Given:</strong><br>
• Unit 2 experienced a Large Break LOCA.<br>
• The crew is performing actions per 2-EOP-TRIP-1, Reactor Trip or Safety Injection.<br>
• The crew is at step 9, Containment Spray Actuation Verification.<br>
• BOTH trains of Phase B/CS have actuated.<br>
• Containment Pressure is 25 psig and rising.<br>
• NO Containment Spray (CS) Pumps are running.<br>
• 2C 4KV Vital Bus is de-energized on bus differential protection.<br><br>
Which ONE of the following action(s) is/are required to start a CS Pump?
</div>

<div style="margin-bottom:14px;">
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>A.</strong> Start 21 CS Pump by depressing the start pushbutton ONLY.</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>B.</strong> Start 22 CS Pump by depressing the start pushbutton ONLY.</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>C.</strong> Block and reset its associated SEC, then start 21 CS Pump using the keyswitch.</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>D.</strong> Block and reset its associated SEC, then start 22 CS Pump using the keyswitch.</div>
</div>

<details>
<summary style="cursor:pointer;color:#60a5fa;font-size:12px;font-weight:600;padding:8px 0;">&#9654; Show Answer &amp; Explanation</summary>
<div style="margin-top:10px;padding:16px;background:#0a0e17;border:1px solid #1a2035;border-radius:6px;font-size:12px;">
<div style="padding:8px 12px;background:#052e16;border-left:3px solid #22c55e;border-radius:0 4px 4px 0;margin-bottom:10px;color:#c8cdd5;"><strong style="color:#22c55e;">&#10003; C. Correct.</strong> The operator must evaluate the plant conditions and determine that the SECs have not been reset. 21 CS Pump is powered from 2A 4KV Vital Bus. Since SECs actuated, the operator must first regain control by blocking and resetting 2A SEC only, then the operator must use the key switch to start 21 CS Pump.</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">&#10007; A.</strong> Incorrect. 2A SEC must be blocked and reset first then the keyswitch must be operated to start 21 CS Pump. Plausible if the operator believes that since CS has actuated that starting the pump is only required.</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">&#10007; B.</strong> Incorrect. 22 CS Pump is powered from 2C 4KV Vital Bus which is locked out on bus differential. Plausible if the operator believes that since CS has actuated that starting the pump is only required.</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">&#10007; D.</strong> Incorrect. 22 CS Pump is powered from 2C 4KV Bus which is locked out on bus differential.</div>
