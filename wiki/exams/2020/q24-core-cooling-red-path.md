<span style="color:#60a5fa;font-size:14px;font-weight:600;">Q24 — Core Cooling Red Path Validation</span>
<span style="font-size:10px;padding:2px 8px;background:#0d1220;color:#6b7280;border-radius:3px;">EPE74G2.1.19 (3.9)</span>
</div>

<div style="font-size:13px;color:#9ca3af;margin-bottom:14px;line-height:1.7;">
<strong>Given:</strong><br>
&bull; Unit 2 has experienced a LOCA.<br>
&bull; The crew is implementing the Critical Function Status Trees.<br>
&bull; The RO has identified that the SPDS Overview Screen indicates a Red Path for Core Cooling.<br>
&bull; RCS Subcooling is &lt; 0 &deg;F.<br>
&bull; No RCPs are running.<br><br>
Which ONE of the following RCS Parameters would validate a Core Cooling Red Path?
</div>

<div style="margin-bottom:14px;">
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>A.</strong> Three (3) hottest CETs reading 751 &deg;F &ndash; 755 &deg;F, with RVLIS Upper Range reading 38 %.</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>B.</strong> Five (5) hottest CETs reading 701 &deg;F &ndash; 705 &deg;F, with RVLIS Full Range reading 39 %.</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>C.</strong> Three (3) hottest CETs reading 1200 &deg;F, with RVLIS Upper Range reading 44 %.</div>
<div style="padding:10px 14px;border-left:3px solid #1a2035;margin-bottom:4px;font-size:12px;color:#c8cdd5;background:#0d1220;border-radius:0 4px 4px 0;"><strong>D.</strong> Five (5) hottest CETs reading 750 &deg;F &ndash; 800 &deg;F with RVLIS Full Range reading 43 %.</div>
</div>

<details>
<summary style="cursor:pointer;color:#60a5fa;font-size:12px;font-weight:600;padding:8px 0;">&#9654; Show Answer &amp; Explanation</summary>
<div style="margin-top:10px;padding:16px;background:#0a0e17;border:1px solid #1a2035;border-radius:6px;font-size:12px;">
<div style="padding:8px 12px;background:#052e16;border-left:3px solid #22c55e;border-radius:0 4px 4px 0;margin-bottom:10px;color:#c8cdd5;"><strong style="color:#22c55e;">&#10003; B. Correct.</strong> Five CETs are &gt; 700&deg;F and RVLIS Full Range is reading &le; 39%.</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">&#10007; A.</strong> Plausible because the first part is correct, five CETs are &gt; 700&deg;F. The candidate may believe that CETs need to be &gt; 750&deg;F as well. The second part is plausible as RVLIS Full Range reading &le; 39% would be correct, however it says &ldquo;Upper Range&rdquo;. Incorrect, because the bases states that &ldquo;the Upper Range is not applicable for use in accessing core cooling status since it only provides indication from the hot leg piping connection to the top of the reactor vessel.&rdquo;</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">&#10007; C.</strong> Plausible because the candidate may believe that for inadequate core cooling, only three (3) CETs need to be &ge; 1200&deg;F. Incorrect because five (5) CETs need to be &gt; 1200&deg;F. Also plausible because, although the stem says RCPs are out of service, the candidate may remember that &le; 44% RVLIS Dynamic Range results in a Purple Path of degraded core cooling being met. Upper Range is also incorrect, see choice &ldquo;A&rdquo; discussion.</div>
<div style="padding:8px 12px;border-left:3px solid #f87171;border-radius:0 4px 4px 0;margin-bottom:6px;color:#9ca3af;"><strong style="color:#f87171;">&#10007; D.</strong> Plausible because the first part is correct, five CETs are &gt; 700&deg;F. The candidate may believe that CETs need to be &ge; 750&deg;F as well. The second part is plausible because Full Range Level is correct and the value of 43% is below the existing dynamic level of 44%. Incorrect because RVLIS Full Range Level needs to be &le; 39% for a Red Path. This would be a Purple Path.</div>
