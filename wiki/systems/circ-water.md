# Circ Water

## Function

Circulating water system provides cooling water to the main condensers from the Delaware River.

<div class="callout callout-exam">
<div class="callout-label">Exam — 2020 Q64</div>
Per AB.CW-0001 Step 3.10: with loss of circulators in the same condenser shell causing condensate pump oscillations (low hotwell levels), <span class="hi-exam">initiate a load reduction to ≤ <span class="val-alarm">83%</span> Reactor Power IAW AB.LOAD-0001 to prevent flashing in the Condensate System</span>. <span class="hi-trap">AB.CN-0001 requires ≤ 85%, but AB.CW-0001 requires the more restrictive ≤ 83%. Always use the directing procedure's limit.</span> Liquid waste releases from CVCS Monitor Tanks via 21 CCW HX are directed to the <span class="hi-exam">Unit 1 Circulating Water (11A&B) Pump discharge</span> — not Unit 2 CW pumps — so loss of Unit 2 circulators does NOT require terminating the release.
